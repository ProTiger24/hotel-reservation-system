# 🏨 Hotel Reservation System

A professional hotel reservation system with dynamic frontend and backend integration.

## ✨ Features

- ✅ Room Management (Standard, Deluxe, Suite)
- ✅ Dynamic Search with Date & Guest Filter
- ✅ Booking System with Real-time Availability
- ✅ bKash Fake API Integration
- ✅ Card Payment Fake API (Stripe-like)
- ✅ Bank Transfer Fake API
- ✅ Professional UI/UX
- ✅ Responsive Design
- ✅ REST API Backend
